\# Projeto SQL - Análise Bancária



\## Sobre o Projeto



Este projeto foi desenvolvido com o objetivo de praticar e demonstrar habilidades em SQL através da análise de um banco de dados bancário fictício.



O banco de dados contém informações sobre clientes, contas, agências e transações financeiras, permitindo a realização de consultas e análises de negócio utilizando conceitos fundamentais e intermediários de SQL.



\## Objetivos



\* Praticar modelagem de banco de dados relacional.

\* Trabalhar com consultas SQL de nível intermediário.

\* Utilizar JOINs para relacionar tabelas.

\* Aplicar funções de agregação como COUNT, SUM e AVG.

\* Utilizar subqueries para análises mais complexas.

\* Introduzir o uso de Window Functions através de rankings.



\## Estrutura do Banco de Dados



O banco é composto pelas seguintes tabelas:



\* clientes

\* contas

\* agencias

\* transacoes



\### Relacionamentos



\* Um cliente pode possuir uma ou mais contas.

\* Uma agência pode administrar várias contas.

\* Uma conta pode possuir várias transações.



\## Consultas Desenvolvidas



Durante o projeto foram realizadas consultas envolvendo:



\* Listagem e filtragem de clientes.

\* Ordenação de dados.

\* Agrupamentos com GROUP BY.

\* Cálculo de médias e totais.

\* JOINs entre múltiplas tabelas.

\* Identificação de clientes com saldo acima da média.

\* Identificação de clientes sem transações.

\* Ranking de clientes por saldo.

\* Análises de movimentação financeira por cliente e agência.
